Lagrange Interpolation

Fitting the unique lowest-degree polynomial that passes exactly through a discrete set of known coordinate points.

Overview & Philosophy 02 / 05

The Foundation Strategy

What it computes: A deterministic method to map and evaluate values falling between target thresholds when an underlying mathematical formulation is missing or non-trivial.

Example: Plotting mid-point metric trends based solely on timestamped industrial sensor readouts.

Mathematical Intuition If we have a set of coordinates, Lagrange interpolation constructs individual polynomial weights for each point. When added together, they create one definitive curve passing perfectly through every coordinate pair.

Formulas & Rules 03 / 05

Interpolation Formulas

Linear Interpolation (2 Points — Degree 1)

f(x) = (x − x₁)(x₀ − x₁)·y₀ + (x − x₀)(x₁ − x₀)·y₁

Quadratic Interpolation (3 Points — Degree 2)

f(x) = (x−x₁)(x−x₂)(x₀−x₁)(x₀−x₂)·y₀ + (x−x₀)(x−x₂)(x₁−x₀)(x₁−x₂)·y₁ + (x−x₀)(x−x₁)(x₂−x₀)(x₂−x₁)·y₂

Step-By-Step Mechanics 04 / 05

Worked Proofs

Example 1

Find y at x = 2 given (1, 2) & (3, 4)

y= (2−3)(1−3)·2 + (2−1)(3−1)·4
y=−1−2·2 + 12·4 = 3
Resulting target: y = 3
Example 2

Find y at x = 5 given (9, 2) & (3, 10)

y= (5−3)(9−3)·2 + (5−9)(3−9)·10
y=446 = 7.33
Resulting target: y ≈ 7.33
